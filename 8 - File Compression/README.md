# 8 - File Compression
 
1. **Archive files**

```
tar -cvf archive.tar files_to_archive/

tar -xvf archive.tar
```

2. **Compress and decompress files**

```
gzip file.txt

gunzip file.txt.gz
```

3. **Create and extract zip archives**

```
zip -r archive.zip directory_to_compress/

unzip archive.zip
```
